Key differences between Google Docs and html documents

Key reasons why Google Docs and html are very different. Firstly, Google Docs gives you access to colors, font and style through a drop down menu. Whereas html you have to input all styles into it. Secondly, Google Docs allows multiple people to work on a document at once, but html does not. Lastly, Google docs is easy to understand and operate. Html is so difficult some days!
